## About Me

- 通信工程背景，关注嵌入式 Linux、端侧 AI、多媒体处理和人机交互系统。
- 喜欢把单点 Demo 做成可以运行、可以展示、可以部署的完整产品原型。
- 主要实践方向包括语音交互、Qt 大屏界面、摄像头采集、硬件编解码、边缘推理和流媒体推送。
- 近期在整理 `friday_voice_speaker`：一个融合智能音箱、Qt 桌面、视觉问答、摄像头、人脸识别和端侧视频处理的综合项目。
